TL;DR
-----

1. Clone the repo and install

.. code:: bash

          $ git clone https://github.com/JnyJny/Geometry3.git
          $ cd Geometry3
	  $ pip3 install -r requirements.txt
	  
1. Install with pip

.. code:: bash
	  $ pip3 install -U geometry3
	  

Uninstall with pip:

.. code:: bash

	  $ pip3 uninstall geometry3

Usage
-----

Once installed, 

.. code:: python

	from Geometry3 import Point

	o = Point()
	p = Point(1,1,1)
	o.is_origin == True
	p.is_origin == False
	p.distance(o)
	p.midpoint(o)
